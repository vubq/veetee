# Monocypher

Vendored Monocypher `4.0.3` sources used only for detached Ed25519 release
signature verification on the ESP32-S3 and in firmware host tests.

Upstream: <https://github.com/LoupVaillant/Monocypher/releases/tag/4.0.3>

The upstream `LICENSE.md` is retained unchanged. Production signing keys are
not part of this component or repository.
